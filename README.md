## Overview
Snake AI trained on reinforcement learning using a DQN based on [pong_RL](https://github.com/wh33ler/QNet_Pong).
## Dependencies


* TensorFlow
* OpenCV (cv2)
* Pygame

## Usage 

```
python snaike.py
```
